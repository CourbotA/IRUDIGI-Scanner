# IRUDIGI-Scanner
PING TSE Génération de Scanner synthétique à partir d'IRM et intelligence artificielle Le but du projet est de parvenir à générer des images de Scanner à partir d'IRM et des dernières avancées en intelligence artificielle. Une piste serait l'utilisation de GAN (generative adversarial network) qui ont donné d'excellents résultats ces dernières années, mais l'idée serait de comparer différentes techniques de Deep Learning afin d’identifier celles pouvant donner les meilleurs résultats
